Integration Guide

To keep your file links safe and private from unauthorized users, you must load images via the backend secure streaming route.

Secure Method

Use this code snippet in your website. Only your web domain with the matching key can retrieve these images:

<!-- Include this in your website -->
<img 
  src="http://localhost:3000/api/image/YOUR_FILE_ID?key=YOUR_SECURITY_KEY" 
  alt="Secure Website Resource"
  style="max-width: 100%; border-radius: 8px;"
/>

How it works:

  • The browser makes a request directly to the secure stream.
  • The server validates the security key match.
  • The server performs a Referer origin match checks.
  • If validation succeeds, the server fetches the media from Telegram on-the-fly and streams it to the page without ever exposing Telegram URLs.
